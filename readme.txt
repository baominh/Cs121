Utilities.java: Write a method that reads in a text file and returns a list of the tokens in that file. Write a method to print out frequency results.

WordFrequencyCounter.java : Count the total number of words and their frequencies in a token list. 

TwoGramFrequencyCounter.java : A 2-gram is two words that occur consecutively in a file. For example, "two words", "words that", and "that occur" are all 2-grams from the previous sentence.
Count the total number of 2-grams and their frequencies in a token list.
